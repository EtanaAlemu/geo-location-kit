Android · Kotlin · Compose

Geo Location Kit

Offline countries, states, cities, and phone country codes for Android — Room SQLite in your APK, optional searchable Compose pickers, 21 languages.

View on GitHub Documentation Releases

Features

Offline data

Pre-built Room database from dr5hn — no JSON parsing at runtime.

Compose pickers

Country, cascading state/city, language, and phone code UI.

Full & lite

Full DB with cities & FTS, or a ~800 KB lite variant.

Localization

English, app locale, or fixed BCP 47 tags from dr5hn translations.

Install (JitPack)

// settings.gradle.kts
maven { url = uri("https://jitpack.io") }

// app/build.gradle.kts
implementation("com.github.EtanaAlemu:geo-location-kit:1.0.0:geo-compose@")
